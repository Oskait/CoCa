# CoCa - Concentration Calculator

A simple Streamlit application to calculate dilutions and manage compounds.

## Features

*   Calculate the required volume of a stock solution for a desired final concentration.
*   Manage a database of compounds.

## Installation & Usage

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/CoCa---Concentration-Calculator.git
    cd CoCa---Concentration-Calculator
    ```

2.  **Install the dependencies:**
    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the application:**
    ```bash
    streamlit run Dilution_Calc_streamlit.py
    ```
